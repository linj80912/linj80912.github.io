1. 伺服器總則

1.1 本伺服器專為 Minecraft 跑酷玩家打造，提供公平競技與良好的遊戲體驗。

1.2 進入伺服器即代表您已閱讀並同意本規範。本伺服器保留隨時變更本規範的權利，所有變更將公告於官方平台，玩家有義務查閱最新內容。

1.3 管理團隊保留最終解釋權，並有權視情況調整規範。

1.4 伺服器內所有地圖、建築與內容均受著作權保護，禁止未經許可擅自使用、修改或發佈。

1.5 若玩家違反本規範並造成法律責任，伺服器管理團隊不承擔任何連帶責任，並將依情節嚴重性決定是否報警處理。

1.6 本伺服器僅提供遊戲服務，不對玩家在遊戲內外的行為負責。所有玩家應遵守當地法律，若有違法行為，需自行承擔責任，本伺服器將依法配合相關單位調查。

2. 玩家行為規範

2.1 禁止騷擾、歧視、謾罵、惡意攻擊或霸凌其他玩家。

2.2 禁止洗版、無意義訊息、重複發送內容或過度使用特殊符號影響其他玩家。

2.3 嚴禁推廣其他伺服器、Discord 群組、YouTube 頻道或與本伺服器無關的內容。

2.4 禁止利用遊戲漏洞（Bug）或特殊機制來獲取不正當優勢，若發現問題請回報管理員。

2.5 禁止下載、擷取或還原伺服器地圖，或使用任何工具來複製關卡內容，亦不得轉售、發佈或擅自改編伺服器內的建築與遊戲資源。

2.6 禁止任何形式的詐騙，包括但不限於冒充管理員、販售虛假商品、詐取他人遊戲財產或個人資料。若發現詐騙行為，伺服器有權永久封鎖並報警處理。

2.7 禁止任何涉及現金或虛擬財產（如遊戲帳號、物品、點券等）的交易，無論是否經由第三方平台。若違反此規定，伺服器將不負責因交易產生的任何損失，並有權封鎖帳號。

3. 技術規範

3.1 玩家可以使用不影響公平性的 Mod，例如 OptiFine、LabyMod 或 Minimap。

3.2 禁止使用任何影響遊戲公平性的模組或插件，包括但不限於飛行、加速、自動跳躍、連點、
    連續衝刺或修改物理引擎來影響跑酷機制的模組或插件。

3.3 禁止對伺服器發起 DDoS 攻擊、惡意發送大量封包或執行過量請求導致伺服器效能下降。

3.4 禁止利用遊戲漏洞或 Bug 影響伺服器穩定性，若發現問題應立即回報管理員。

4. 建築師規範

4.1建築師不得擅自修改或刪除伺服器內建築，所有變動須經管理團隊批准。

4.2請避免使用會影響伺服器穩定的指令或工具。

4.3所有地圖應為原創或取得授權，抄襲將受處罰。

5. 違規處罰

5.1 洗版、騷擾、發送垃圾訊息或不當言論，將視情況禁言或暫時封鎖帳號。

5.2 濫用介伏盒 Bug 或利用方塊特性干擾其他玩家，將視情況進行懲處。

5.3 任何形式的作弊（如外掛），首次封鎖 30 天，二次則永久封鎖。

5.4 建築師違反規範，將視情況進行警告、撤銷建築權限或永久封鎖帳號。

5.5 任何嚴重違規行為，如詐騙、濫用漏洞、惡意破壞伺服器，將被永久封鎖，管理團隊保留最終決定權。

6. 申訴機制

6.1 若認為處罰不公，可透過 Discord 論壇提交申訴，並附上相關證據，如影片或聊天紀錄。

6.2 管理團隊將依據證據進行審核，並擁有最終決定權，必要時可調整處罰結果。

6.3 申訴應在違規處罰後 7 天內提出，逾期不予受理。

6.4 申訴過程中，玩家應遵守基本禮儀，惡意謾罵或騷擾管理員者，將可能導致申訴失敗或進一步懲處。

7. 附則

7.1 本規範可能會依伺服器狀況調整，所有變更將提前公告。

7.2 若遇到未明確規範的情況，管理團隊將依公平性與社群利益來判斷處理。

7.3 任何試圖規避規範的行為仍將受到懲處，管理團隊擁有最終決定權。

7.4 若因技術問題、硬體故障、不可抗力因素或其他原因導致伺服器暫時關閉或永久終止，管理團隊不承擔任何責任，也不提供遊戲數據備份、補償或退款。

1. 總則

1.1 本規範適用於所有擁有建築權限的玩家，包括正式建築師、實習建築師與特許玩家，確保伺服器地圖標準與公平性。

1.2 所有建築師需遵守此規範，並隨時關注更新。

2. 地圖規範

2.1 每張地圖必須具備中英文名稱，用於讓玩家可識別及排行榜顯示使用。

2.2 地圖必須標示清楚的難度，包括簡單、普通、困難或極限等等難度，並符合伺服器的挑戰標準。

2.3 起點與終點必須使用烽火台，確保統一性與易辨識性。

2.4 所有地圖需經過測試，確保每個跳躍皆可通過，且無法利用漏洞跳關。

2.5 非段位形式的地圖應設計合理的檢查點，避免過度懲罰性設計，影響遊戲體驗。

2.6 應提供明確的視覺指引，例如地毯、燈光或標誌，避免玩家迷路。

3. 地圖提交與審核

3.1 地圖完成後，建築師需提交地圖名稱（中英文）、難度分類、起點與終點座標，以及特色與機關說明。

3.2 管理團隊將於 7 天內審核地圖，確保其符合規範並具備公平的挑戰性。

3.3 若地圖未通過審核，建築師需根據回饋進行修改，直至符合標準。

4. 建築師行為規範

4.1 建築師不得擅自更改伺服器內建築，包括已上線的地圖、伺服器大廳或其他玩家的作品，所有變動須經過管理團隊或地圖作者批准。

4.2 禁止使用指令刪除已上線的地圖，所有地圖移除須經過管理團隊批准。

4.3 禁止使用指令執行可能導致伺服器崩潰或重啟的操作。

4.4 禁止濫用 WorldEdit 或其他工具，若有大範圍修改需求須先與管理團隊確認。

4.5 禁止利用權限進行惡意行為，例如強制傳送玩家、生成大量實體或影響遊戲體驗的操作。

4.6 請尊重著作權，禁止抄襲其他作者的作品，所有地圖應為建築師原創或取得授權後進行改編。

4.7 地圖正式上線後，若發現 Bug 或難度不平衡，建築師應主動回報並協助修正。

5. 違規處罰

5.1 擅自修改或刪除地圖將被警告，並撤回相關變更。再次違規者將被限制建築權限七天，累犯者將被移除建築師資格。

5.2 濫用指令導致伺服器問題將受到警告，如再次發生，將限制建築權限十四天，嚴重者將被永久移除資格。

5.3 利用權限進行惡意行為，如強制傳送玩家或干擾遊戲體驗，將直接限制建築權限七天，嚴重者將被封禁帳號。

5.4 若建築師的行為導致伺服器崩潰或停止運行，將立即撤銷建築權限，並根據情節嚴重性決定是否永久封禁帳號。

5.5 抄襲其他作者的作品經查證屬實將導致警告，並要求立即撤回相關地圖。若再次違規，將限制建築權限七天，嚴重者將被移除建築師資格。

6. 建築師晉升與降級

6.1 實習建築師(建築師e)需提交至少三張合格地圖，並經審核通過，方可晉升正式建築師。

6.2 正式建築師需提交至少三張優秀地圖，且具備指導新建築師的能力，方可晉升高級建築師(建築師pro)。

6.3 若建築師超過一個月未貢獻地圖，管理團隊可視情況降級或撤銷其權限。

6.4 若建築師違規次數達三次以上，將被永久移除建築師資格。

7. 附則

7.1 本規範自公告日起生效，所有建築師皆須遵守，若不同意規範內容，請勿申請建築師職位。

7.2 本規範最終解釋權歸伺服器管理團隊所有。
